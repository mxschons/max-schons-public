## Lab — continuously (roughly yearly)

| Value | Description | Reference range |
|-------|-------------|-----------------|
| ApoB | | Ideally < 40 |
| Lipid Panel | Total cholesterol, LDL, HDL, triglycerides, oxidized LDL, HS-OMEGA index | LDL < 70 |
| OGTT with insulin (every 2–5y) | Fasting, then every 30 min for 2h. 3 days prior: ~100–150g carbs/day (no low/no-carb). | See OGTT ranges below. |
| ALT | Liver | Peter Attia: < 20 (standard reference: 7–55 U/I male, 7–45 female) |
| hs-CRP | | |
| Holotranscobalamin | B12 marker | Standard lab ranges |
| Cystatin C | Best kidney marker | |
| Hormone panel | Total testosterone, SHBG, free androgen index, Vermeulen free testosterone (get 2–3 fasting measurements), LH, FSH, estradiol, prolactin | |
| Vitamin D | Important for many things | > 50 ng/mL |
| Continuous Glucose Monitor (every 2–3y) | For 2–8 weeks | Average < 100, SD < 15 |

**OGTT reference ranges (per Peter Attia):**

- Glucose: fasting 60–100 mg/dL (Peter likes < 90); 1h < 200 (Peter likes < 120–130, depending on muscle mass); 2h < 140 (Peter likes < 100). You can be in these ranges and still have postprandial hyperinsulinemia.
- Insulin: fasting < 25 mIU/L (Peter likes < 6); 1h 18–276 (Peter likes < 30); 2h 16–166 (Peter likes < 20).

Peter Attia's full panel with optimal / low / high columns is in his [AMA series](https://peterattiamd.com/ama01/).

## Physical testing

Most of these should be done roughly yearly (6–24 months).

| Value | Description | Reference range |
|-------|-------------|-----------------|
| Blood pressure | Standard medical protocol for 2 weeks morning + evening. Every ~2y a 24h measurement. | < 120/80 |
| Resting heart rate | Any wearable | < 60 |
| Sleep lab | [Home device](https://amzn.eu/d/aD6XCPw) for the 6–24 month cadence. Every ~5y, proper sleep-lab gear with expert physicians. | Up to 5 events is "normal", but I want 0–1 hypoxia and 0 apnoea events per hour. |
| Skill assessment | 4 things at every joint (ankle, knee, hip): symmetry, stability, slow controlled movement, awareness (e.g. heels down when squatting), full range of motion. | Experienced trainer testing you, or a video review (tight clothes). |
| VO2 max | Wearables approximate (underestimate); proper test is better. | Up to 40y: > 50 (males and females) |
| Grip strength | [Dynamometer](https://amzn.eu/d/7j6fdTX), age-adjusted > 90th percentile. For a 30y male ~65kg. Little left / right difference. | 2-min dead hang + 1-min farmer carry with bodyweight is excellent. |
| Strength (lower body) | Squat with dumbbell | Hold ~half bodyweight at bottom for ~45s |
| DXA full-body scan | Follow up with separate bone-density scan if needed | ALMI 9–10 males / 7–8 females; FFMI > 22 males / 18.5 females |
| Muscular endurance | Push-up with full elbow lockout at top + chest touch at bottom | 25 consecutive (male) / 15 (female) |
| Power tests | Standing jump; vertical jump both hands | >> your body size; > 60cm male / > 50cm female (age-adjusted: > 50y → > 50cm male, > 42cm female) |
| Anaerobic capacity | Heart-rate recovery at max / at peak fatigue | 30 bpm recovery in 60s (~½ beat per second) |
| Brain MRI | (Want to explore more in the future.) | |

Sources: [Peter Attia AMA 14](https://peterattiamd.com/ama14/), [Longevity Office](https://www.longevityoffice.com/), [AMA 27](https://peterattiamd.com/ama27/), [Andy Galpin × Huberman Lab series notes](https://podcastnotes.org/huberman-lab/dr-andy-galpin-how-to-assess-improve-all-aspects-of-your-fitness-huberman-lab-guest-series/). Deep reference: [ACSM's Guidelines for Exercise Testing and Prescription](https://shop.lww.com/ACSM-s-Guidelines-for-Exercise-Testing-and-Prescription/p/9781975150181).

## Lead measures

Weekly inputs for optimal health:

- 3h low-effort cardio (60–80% max HR)
- 1h high-effort cardio (> 80% max HR with breaks)
  - Most established regimen: 4 min at 90–95% max HR + 4 min "rest" at 70–80% HR ([Helgerud et al. 2007](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0113637))
- 3h strength training
- 1h stability / injury prevention / stretching
- One rest week every 3–6 months

Sleep references: [JAMA](https://jamanetwork.com/journals/jama/fullarticle/2798400), [NEJM](https://www.nejm.org/doi/full/10.1056/NEJMcp1816152), [Huberman on sleep (YouTube)](https://www.youtube.com/watch?v=WkPjOmgh40A).

## Health data and timeline

I do a weekly check-in on ongoing and new health issues in a simple Google spreadsheet — one column per symptom I'm watching (e.g. knee pain, wrist pain, tinnitus), one row per week.

All medical records go in a folder as scanned documents. Lab parameters are transferred into a separate sheet for better longitudinal overview.
