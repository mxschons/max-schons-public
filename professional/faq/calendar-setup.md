# How do I set up my calendar? (2025-04)

1. Use [Reclaim.ai](https://reclaim.ai/) to sync personal calendar to professional and vice versa. Professional tiers allow syncing multiple calendars.
2. Use OOO events to signal when you're usually blocked. Easily moved if timezones change; provides buffer.
3. When you want to get something done, schedule it in your calendar — ideally invite someone else to the event.
4. Strongly recommend:
   - At least one meeting-free day every week. Add as a blocker so others can see it.
   - A private weekly check-in on Monday morning to orient yourself.
   - A personal check-in for life-admin and non-work stuff on Friday morning, so you can act during usual office hours.

Note: I also add the time I'll be back to OOO events.
